## How to run this project

```sh
git clone https://github.com/Akash-27022002/user-management.git
cd user-management
```
## How to run Backend
```sh
cd backend
npm i
npm run dev
```

## How to run Backend
```sh
cd frontend
npm i
npm run dev
```

Checkout [package.json](../package.json) for more commands
